Punch Off (2018)

Punch Off is VR parkour game in which the player punches the ground and walls to launch themselves from place to place. For this project I work as a programmer, a level designer, and a 3D modeller. This project is ongoing, and still in development.

Trailer

Saco (2017)

Saco is a JavaScript-based fishing game that I created by myself. I did all of the art and animation for this game as well as the programming. The game focuses on having a very simple yet enjoyable gameplay with a somber tone.

Korean House (2017)

This was a project I did with three other students for our 3D animation class. While contributing to the models and textures, I was also the one who brought all the aspects of the project together in Unity. I also made a particle system for fog.

The Floor is Lava (2017)

This game is a 2D Platformer that I made in a group project with three other students. I was a programmer and level designer for the project. The game's goal is to provide a fun and difficult platforming experience with appealling graphics. The game has very tight controls and focuses on player platforming skill. The project was made as a semester long project for C# coding class.

To Greener Pastures (2017)

To Greener Pastures is a game I made for Hero Jam 2017 along with four other students. I was a programmer on this project, and I focused on the gameplay programming. The game has combat from an overhead view and allows the player to explore a procedurally-generated, post-apocolyptic world. The game takes place in a world in turmoil as one tribe begins to search for a way out of the desert...

Hot Doug (2015)

Hot Doug is a 2D Platformer I made with Sandbag Games. Hot Doug is the story of a hot dog going on an adventure to save a pair of hot dog buns that he's in love with (Note: This game predates Seth Rogen's Sausage Fest). I was the sole programmer and level designer on this project. The game was made in GameMaker: Studio.

About Christian Axisa
